HOW TO INSTALL
==============

For Mod Users:
--------------

Download the latest installer from http://files.minecraftforge.net/ and follow
the instructions given by the installer.


For Mod Developers:
-------------------

Download the latest MDK from http://files.minecraftforge.net/ and follow the
instructions given in the README.txt file.


For Contributors:
-----------------

Follow the instructions given in the Minecraft Forge GitHub wiki:

   https://github.com/MinecraftForge/MinecraftForge/wiki/If-you-want-to-contribute-to-Forge


Requirements:
-------------

For mod devs and contributors:

 - You must have a JDK installed and accessible.

 - If you do not wish to use the gradle wrapper, you can install Gradle from:

      http://www.gradle.org/
